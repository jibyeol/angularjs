왜 TypeScript?
 - 생산성 
   * JavaScript는 컴파일 과정이 없어서 런타임시에만 에러가 발생한다.
   * JavaScript 엔진이 변수의 타입을 예상해도, 개발 도구에서 타입을 알지 못하면 알려주는 정보에 한계가 있다.

Dart를 사용하지 않는 이유
 - 서드 파티 JavaScript 라이브러리와 호환성이 떨어짐
 - Dart로 만단 애플리케이션은 Dartium이라고 하는 특별한 버전의 Chrome 브라우저에서만 동작
 - 컴파일해서 만든 JavaScript 코드의 가독성이 떨어짐
 - Dart 개발자 커뮤니티가 아직 작다.
