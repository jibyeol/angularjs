프레임워크 = 프레임에서 제공하는 설계 구조를 사용하며 그 구조에 어울리는 방식으로 코드를 작성해야한다.
라이브러리 = 기존에 개발하던 코딩 스타일을 그대로 유지하면서 필요한 기능을 골라서 추가한다.

1.1.1 종합 프레임워크 : 웹 애플리케이션 개발에 필요한 모든 기능을 제공
1.1.2 경량 프레임워크 : 특정 기능을 위한 기능만을 제공하는 프레임워크
1.1.3 라이브러리 : 문제 해결이나 특정 기능을 추가하기 위해 만들어졌고, 프레임워크의 사용 여부와 관계없이 자유롭게 쓸 수 있다.
1.1.4 Node.js : 프레임워크, 라이브러리, 실행환경
